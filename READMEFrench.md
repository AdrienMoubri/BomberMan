## Windows
* Commencez par télécharger puis installer votre compilateur [Mingw](https://sourceforge.net/projects/mingw/files/)
* Téléchargez : [SDL 1.2.15](https://www.libsdl.org/release/SDL-devel-1.2.15-mingw32.tar.gz)
* Téléchargez : [SDL Image 1.2](https://www.libsdl.org/projects/SDL_image/release/SDL_image-devel-1.2.12-VC.zip)
* Téléchargez : [SDL TTF](https://www.libsdl.org/projects/SDL_ttf/release/SDL_ttf-devel-2.0.11-VC.zip)
* Ouvrez le fichier SDL-devel-1.2.15-mingw32.tar.gz et copier le dans un dossier facilement accessible
* Ouvrez les zip ouvrez des libraires SDL image et TTF puis ouvrez les dossiers include
* Copier les fichier .h dans le dossier /SDL-1.2.15/include/SDL
* Ouvrez le dossier lib et copier tout le contenue dans le dossier /SDL-1.2.15/lib
* Définir la variable d'environnement SDLDIR avec le chemin du dossier /SDL 1.2.15.
